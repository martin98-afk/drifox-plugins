# teach 🎓

> AI 私人教师技能 — 让 AI 当你的私人老师

基于学习目标进行多轮交互式教学，自动管理工作区（MISSION.md、lessons/、learning-records/、reference/），跟踪学习进度，持续教会你任何知识。

## 功能

- **目标驱动** — 先明确学习动机和目标，建立学习契约
- **最近发展区** — 根据已有知识，教下一个刚好能学会的内容
- **输出即学习** — 每节课生成自包含 HTML 课程文件
- **记录沉淀** — 学习记录像 ADR 一样沉淀关键洞见，跨会话延续
- **教学素材** — 生成参考速查表和术语表，方便打印回顾

## 使用方法

告诉 AI 想学什么即可激活：

- "教我 Python 列表推导式"
- "我想学 Kubernetes"
- "帮我讲解 React 的 useEffect"
- "teach me git rebase"

AI 会先建立学习契约（了解你的水平和目标），然后按节奏一步步教学。

## 工作区结构

教学工作区在**当前目录**下自动创建：

```
./                        # 教学工作区根目录
├── MISSION.md            # 学习目标与动机
├── RESOURCES.md          # 参考资源列表
├── reference/            # 速查表（可打印 HTML）
├── lessons/              # 每节课的 HTML 输出
├── learning-records/     # 学习记录（关键洞见）
└── assets/               # 共享 CSS/JS
```

## 快捷键

| 按键 | 功能 |
|------|------|
| `教我XXX` / `我想学XXX` | 激活教学流程 |
| `懂了` / `太快了` | 调整教学节奏 |
| `继续` | 从上次中断处继续 |
