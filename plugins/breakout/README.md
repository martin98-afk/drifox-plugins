# 🏓 Breakout - DriFox 打砖块插件

在 DriFox 聊天界面中直接玩经典打砖块！

## 使用

输入 `/breakout` 打开打砖块卡片。

## 功能

- **鼠标控制**：移动鼠标控制挡板左右移动
- **键盘控制**：方向键或 A/D 键控制挡板，空格键发射球，R 键重新开始
- **计分系统**：击碎不同颜色的砖块获得不同分数
  - 红色砖块：50分
  - 橙色砖块：40分
  - 黄色砖块：30分
  - 绿色砖块：20分
  - 蓝色砖块：10分
- **生命值**：3条生命，球落底即失去一条命
- **胜利条件**：击碎所有砖块
- **失败条件**：生命耗尽
- **砖块颜色分层**：从顶到底呈红橙黄绿蓝渐变
- **跟随 DriFox 主题色**

## 操作说明

| 操作 | 输入 |
|------|------|
| 移动挡板 | 鼠标移动 / 方向键 / A D |
| 发射小球 | 点击画布 / 空格键 |
| 重新开始 | R 键 / 点击重新开始按钮 |
