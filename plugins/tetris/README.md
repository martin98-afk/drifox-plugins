# 🧱 Tetris - DriFox 俄罗斯方块插件

在 DriFox 聊天界面中直接玩经典俄罗斯方块！

## 使用

输入 `/tetris` 打开俄罗斯方块卡片。

## 功能

- **7 种标准方块**: I / O / T / S / Z / L / J
- **方向键操作**: ← → 移动，↑ 旋转
- **加速下落**: ↓ 软降（每格+1分），空格 硬降（+2分/格）
- **下一块预览**: 侧边栏实时显示
- **幽灵方块**: 落点位置半透明预览
- **计分系统**: 单行100分，双行300，三行500，四行800（Tetris）
- **等级递增**: 每消除10行升一级，速度加快
- **P 键暂停/继续**
- **跟随 DriFox 主题色**

## 操作说明

| 按键 | 功能 |
|------|------|
| ← / → | 左右移动 |
| ↑ | 顺时针旋转（带 wall kick） |
| ↓ | 软降（加速下落） |
| 空格 | 硬降（直接落底） |
| P | 暂停 / 继续 |
